> 本專案使用 [IONIC](http://ionicframework.com/) 製作

主要功能
========
1. 接近班次顯示 (倒數計時以分鐘為單位)
2. 靜態時刻表
3. 中英文介面

HACK
====

```
$ git clone git@github.com:imZack/tnua-bus.git
$ npm install
$ ionic build android
$ ionic run android
```

enjoy!

免責聲明
========
- 實際公車行駛時間請以最新公告為主
- 本專案不代表台北藝術大學
